# Supply Chain Analytics

SQL-based supply chain analytics projects built from real-world experience 
supporting pharmaceutical supply chain planning and execution.

## Live Demo

Safety Stock Calculator:
https://ketki-pawar27.github.io/supply-chain-analytics/safety_stock_calculator.html

## Projects

### 1. Safety Stock Calculator (Live Demo above)
Interactive calculator to compute optimal safety stock levels using 
demand variability, lead time deviation, and service level targets.

### 2. Inventory Coverage Analysis (MOH)
SQL queries to monitor months on hand across SKUs and identify coverage risks.

### 3. S&OE Exception Reporting
SQL-based demand vs supply gap analysis for short-term execution planning.

### 4. OTIF Performance Tracking
Track On Time In Full delivery performance by supplier and product.
