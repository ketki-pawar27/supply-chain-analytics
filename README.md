# Supply Chain Analytics

SQL-based supply chain analytics projects built from real-world experience supporting pharmaceutical supply chain planning and execution.

## Projects

### 1. Safety Stock Calculator
Calculate optimal safety stock levels using demand variability and lead time data.

### 2. Inventory Coverage Analysis (MOH)
Monitor months on hand across SKUs to identify coverage risks.

### 3. S&OE Exception Reporting
Identify demand vs supply gaps for short-term execution planning.

### 4. OTIF Performance Tracking
Track On Time In Full delivery performance by supplier and product.

## Tech Stack
- SQL (Advanced)
- Amazon QuickSight
- AWS Analytics
- Databricks / Starburst
